# 🥁 Drum Kit Project

A simple and interactive Drum Kit web application built using **HTML, CSS, and JavaScript**.  
Click the buttons or press keyboard keys to play different drum sounds.

---

## 🚀 Live Demo
👉 https://anilsingh0070.github.io/drumInstrument/

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM + Events)

---

## 🎮 Features
- Click-based drum sounds
- Keyboard key support
- Multiple drum sounds (tom, snare, kick, crash etc.)
- Simple and responsive UI

---

## 📁 Project Structure

drumInstrument/
│── index.html
│── styles.css
│── index.js
│
├── images/
│ └── (drum images)
│
└── sounds/
└── (drum sound files)

---

## 🎹 How It Works
- Each button is linked with a specific sound
- JavaScript listens for **click and keyboard events**
- When triggered, corresponding audio plays
